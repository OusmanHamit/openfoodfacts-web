Inteligența din spatele aplicațiilor și API-ului Open Food Facts

Open Food Facts colectează date utilizând un API WRITE, le prelucrează și le pune la dispoziție folosind un API READ și o versiune web. Calcularea Nutri-Score, grupurile NOVA, extragerea aditivilor, indiferent de limbaj, sinonime sau potențiale greșeli de ortografie. Aceasta este doar o mică parte din subiectele abordate de echipa de server, pentru a asigura transparența alimentelor la scară planetară.

Contribuie la Nucleu folosind Perl

Vrei să înveți o nouă limbă? Serverul este în principal codificat în Perl. Indiferent dacă ești un dependent de Perl experimentat sau tocmai ai atins Perl pentru sarcini de administrator de sistem, există o eroare de dimensiunea ta și mentori care te pot ghida. Și, de asemenea, mai multe moduri de a face asta ;-)

Creați instrumente utile în orice limbă

HungerGames este exemplul unui mic joc codificat în reactie care are un impact pozitiv asupra Open Food Facts. LanguageSwitcher, un alt instrument, este codificat în Python. Instrumentul dvs., limba dvs. (vă rugăm să evitați Assembler ;-)
