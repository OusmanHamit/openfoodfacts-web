Kecerdasan di balik aplikasi dan API Open Food Facts

Open Food Facts mengumpulkan data menggunakan WRITE API, memproses dan membuatnya tersedia menggunakan READ API dan versi web. Menghitung Nutri-Score, grup NOVA, mengekstrak zat aditif, terlepas dari bahasa, sinonim, atau kemungkinan kesalahan ejaannya. Ini hanyalah sebagian kecil topik yang dibahas oleh tim peladen, untuk memastikan transparansi makanan sampai pada skala dunia.

Berkontribusi ke Core menggunakan Perl

Ingin belajar bahasa baru? Peladennya terutama dikodekan dalam Perl. Apakah Anda seorang pecandu Perl berpengalaman, atau baru saja menyentuh Perl untuk tugas - tugas sys-admin, ada bug sesuai ukuran Anda, dan mentor untuk membimbing Anda. Dan juga ada lebih dari satu cara untuk melakukannya ;-)

Buat peralatan yang berguna dalam bahasa apapun

HungerGames adalah contoh dari permainan kecil yang dikodekan dalam react yang memiliki dampak positif di Open Food Facts. LanguageSwitcher, peralatan lainnya, dikodekan dalam Python. Peralatan Anda, bahasa Anda (harap hindari Assembler ;-)
